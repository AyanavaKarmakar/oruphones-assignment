## Getting Started

### Backend

1. Clone the repo

```bash
   git clone https://github.com/AyanavaKarmakar/oruphones-assignment.git
```

2. Install dependencies

```bash
    cd backend
   npm install
```

3. Create a .env file and add the following

```bash
   MONGO_URI=<your_mongoDB_uri>
```

4. Run the server

```bash
   npm run dev
```

### Frontend

1. Clone the repo

```bash
   git clone https://github.com/AyanavaKarmakar/oruphones-assignment.git
```

2. Install dependencies

```bash
    cd frontend
   npm install
```

3. Create a .env file and add the following

```bash
   VITE_API_URL=<your_backend_api_url>
```

4. Run the server

```bash
   npm run dev
```
